03 — Agents · Browser automation (MV3)

Fills the form. Never hits submit.

autapply — an autonomous job-application agent that lives in your own Chrome. It discovers live-verified openings at top AI companies and fills Greenhouse, Lever, Ashby and Workday applications in background tabs while you browse.

A stack of application pages hangs in the dark. On the front one, a caret of light types row after row — the fill loop, running live. The submit control never lights up: it stays locked until the page is handed back to you.

04 — Computer vision · Accessibility

Run Windows with your hand.

Gesture Accessibility Controller — webcam, MediaPipe, and a custom two-head neural model (~10k parameters, one ONNX graph) turn hand gestures into desktop control. Built accessibility-first for users with motor impairments; nothing fires until you engage.

That's a hand of light — MediaPipe's 21-landmark skeleton, the exact structure the model reads. Watch it cycle the vocabulary: open palm, fist, point. Fingertips glow saffron; the wrist is the anchor.

05 — Developer tools · Agentic CLI

A terminal with an Indian soul.

sarvam-code — an agentic coding CLI hardened for smaller Indian LLMs. A real plan → act → observe loop that reads, writes and edits files, searches your codebase, runs commands — and replies in Hindi, Tamil, Telugu or Bengali.

The Ashoka Chakra as a tool loop: a spinner of light runs the twenty-four spokes like a REPL heartbeat, and packets stream in and out along them — reads out, writes back, never stopping.

06 — AI infrastructure · MCP

Fourteen tools. One command.

namaste-mcp — a Model Context Protocol server exposing Indian public data as native tools for Claude, Cursor and any MCP client: GSTIN validation, full GST invoice math, IFSC, PIN codes, mandi prices, railways, weather, FX.

Namaste, rendered literally: a lotus of fourteen petals, one per tool, breathing around a single core — the server. Packets run the midribs from petal to center and back. Real offline computation, not just API proxying.
